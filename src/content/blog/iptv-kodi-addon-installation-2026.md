### 2. Install the Addon from a Repository or ZIP File

The installation method depends on how the addon is distributed.

#### From a Repository:

1.  **Go Back to Add-ons Menu:** Return to the main "Add-ons" menu.
2.  **Install from Repository:** Select "Install from repository."
3.  **Choose a Repository:** Select the repository that contains the IPTV addon you want to install (e.g., the official Kodi Add-on repository, or a third-party repository you've already added).
4.  **Find the IPTV Addon:** Navigate through the repository's categories (e.g., "Video add-ons") to find the IPTV addon.
5.  **Install:** Click on the addon and then click the "Install" button. Kodi will download and install the addon.

#### From a ZIP File:

1.  **Download the ZIP File:** Download the ZIP file containing the IPTV addon from a trusted source. Make sure you know where the file is saved on your device.
2.  **Go Back to Add-ons Menu:** Return to the main "Add-ons" menu.
3.  **Install from ZIP File:** Select "Install from zip file."
4.  **Locate the ZIP File:** Browse to the location where you saved the ZIP file.
5.  **Install:** Select the ZIP file and click "OK." Kodi will install the addon from the ZIP file.

### 3. Configure the IPTV Addon

Once the addon is installed, you'll typically need to configure it with your IPTV provider's details.

1.  **Find the Addon:** Go to the "Add-ons" menu and then select "Video add-ons."
2.  **Launch the Addon:** Find the IPTV addon you just installed and click on it to launch it.
3.  **Configuration Settings:** Look for a "Settings" or "Configuration" option within the addon. This is usually accessible via the context menu (right-click or long-press on the addon).
4.  **Enter IPTV Details:** Enter your IPTV provider's details, such as the M3U playlist URL, EPG URL, or username/password. These details should be provided by your IPTV provider.
5.  **Save Settings:** Save the configuration settings.
6.  **Start Streaming:** The addon should now be ready to use. Browse the channel list and start streaming live TV.

## Troubleshooting Common Issues

Even with a careful setup, you might encounter some issues. Here are common problems and how to fix them:

*   **Buffering:** Try switching to a lower resolution stream or ensuring your internet connection is stable. Close other applications that are using bandwidth.
*   **Addon Not Working:** Check for addon updates. Clear Kodi's cache. Reinstall the addon.
*   **No Channels Listed:** Double-check your IPTV provider details in the addon's settings. Ensure your subscription is active.
*   **Error Messages:** Read the error message carefully. Search online forums for solutions related to that specific error.
